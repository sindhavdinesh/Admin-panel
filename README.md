## 🎭 The Motion & Animation Experience

Camioca is not just static boxes. It’s a living, breathing UI built on micro-interactions:

*   **✨ The Hover Kinetic Effect:** Sidebar items and buttons scale up and transform smoothly using specialized CSS timing functions:
    ```css
    transition: all 0.25s cubic-bezier(0.4, 0, 0.2, 1);
    ```
*   **🌀 Elastic Dropdowns:** The account dropdown menu glides down and rotates the Chevron icon 180 degrees in a seamless, fluid motion.
*   **🔍 Borderless Search Focus:** The quick-search field features a responsive line glow transition that activates upon focus without jumping layouts.

---

## 📱 Fluid Responsiveness (320px ➡️ 1920px)

Our layout engine dynamically adapts to any device viewport with zero horizontal overflow:

*   **⚡ Micro-screens (320px - 360px):** Content compresses intelligently; user names collapse into a simple, elegant profile indicator.
*   **⚡ iPhones & Androids (375px - 430px):** Widget layouts flow in a single column with 100% fluid percent width.
*   **⚡ Large Displays & Desktops:** Automatically expands back into a double-column dashboard structure.
